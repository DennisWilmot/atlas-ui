# Social Buttons Example

Disposition: `EXAMPLE_ONLY`

Status: `EXAMPLE_ONLY_DONE`

This directory quarantines social/provider auth button treatment as Storybook-only reference material. It is not part of the package exports and must not be imported into public primitives or patterns.

Provider identity, OAuth routing, analytics, icons, and submitted data belong to the consuming app. This example accepts neutral actions through props and returns `null` when no visible provider actions exist.
