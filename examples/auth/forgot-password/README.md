# Forgot Password Examples

Disposition: `EXAMPLE_ONLY`

Status: `EXAMPLE_ONLY_DONE`

This directory quarantines forgot-password page treatment as Storybook-only reference material. It is not part of the package exports and must not be imported into public primitives or patterns.

The local example component is prop-driven so states can be documented without auth logic, reset routing, or account records.

Example data stays neutral: no fake users, fake emails, fake URLs, hardcoded metrics, demo invite flows, reset links, or app-specific routes.
