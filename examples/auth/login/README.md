# Login Page Examples

Disposition: EXAMPLE_ONLY.

These Storybook examples are documentation-only login page compositions. They are not exported from the package and must not be imported by `src/primitives/` or `src/patterns/`.

The app must own routing, authentication providers, recovery behavior, and submitted data.

Social/provider buttons are represented only as neutral Storybook examples. Real provider names, icons, OAuth routing, and analytics stay in the consuming app.
