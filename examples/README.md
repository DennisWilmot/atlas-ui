# Atlas UI Examples

Examples are documentation-only compositions. They are never exported from the package and must not be imported by public primitives or patterns.

- `auth/forgot-password/` keeps forgot-password pages quarantined as EXAMPLE_ONLY.
- `auth/signup/` keeps signup pages quarantined as EXAMPLE_ONLY.
- `auth/social-buttons/` keeps social/provider auth buttons quarantined as EXAMPLE_ONLY.
- `assets/illustrations/` keeps neutral illustration assets quarantined as EXAMPLE_ONLY.
- `dashboards/` keeps dashboard pages quarantined as EXAMPLE_ONLY.
- `navigation/headers/` keeps header navigation layouts quarantined as EXAMPLE_ONLY.
- `sections/headers/` keeps section-header compositions quarantined as EXAMPLE_ONLY.
- `not-found/` keeps not-found section treatments quarantined as EXAMPLE_ONLY.
